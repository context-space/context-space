# 隐私政策

> **重要提示**：本中文版本仅供参考，如有任何冲突或歧义，请以 [英文版本](https://context.space/privacy) 为准。
>
> **Important Notice**: This Chinese version is for reference only. In case of any conflicts or ambiguities, the English version shall prevail.

**最后更新时间：2025年7月1日**

## 1. 引言

Context Space 致力于保护您的隐私并确保您个人信息的安全。本隐私政策说明了当您使用我们的上下文工程基础设施平台时，我们如何收集、使用、共享和保护您的信息。

通过使用我们的服务，您同意按照本隐私政策所述收集和使用您的信息。如果您不同意我们的隐私做法，请不要使用我们的服务。

## 2. 我们收集的信息

### 2.1 您直接提供的信息
- **账户信息**：姓名、电子邮件地址、用户名和密码
- **个人资料信息**：您选择提供的其他个人资料详情
- **配置数据**：集成设置、偏好设置和自定义配置
- **支持通讯**：您在联系客户支持时提供的信息

### 2.2 自动收集的信息
- **使用数据**：您如何与我们的平台互动、使用的功能和使用频率
- **技术数据**：IP地址、浏览器类型、设备信息、操作系统
- **日志数据**：服务器日志、错误报告和系统性能指标
- **分析数据**：用户行为模式和平台性能指标

### 2.3 来自第三方集成的信息
当您将第三方服务连接到 Context Space 时，我们可能收集：
- **身份验证令牌**：OAuth 令牌和 API 密钥用于服务访问
- **服务数据**：您授权我们从连接的服务中访问的数据（GitHub 仓库、Slack 消息、Airtable 记录等）
- **集成元数据**：有关您连接的服务及其配置的信息
- **同步内容**：通过我们的平台在您连接的服务之间流动的数据

### 2.4 Cookie 和跟踪技术
- **必要 Cookie**：平台功能和安全所需
- **分析 Cookie**：帮助我们了解平台的使用情况
- **偏好 Cookie**：记住您的设置和偏好
- **第三方 Cookie**：来自集成服务和分析提供商

## 3. 我们如何使用您的信息

### 3.1 服务提供
- 提供和维护我们的上下文工程平台
- 处理和促进第三方服务之间的集成
- 验证用户身份并管理账户访问
- 启用数据同步和工作流自动化

### 3.2 平台改进
- 分析使用模式以改进我们的服务
- 开发新功能和能力
- 监控平台性能和可靠性
- 进行研究和开发活动

### 3.3 通讯
- 发送重要的服务更新和安全通知
- 回应客户支持询问
- 提供技术协助和故障排除
- 分享相关的产品更新和改进（经您同意）

### 3.4 安全和合规
- 检测和防止欺诈活动
- 维护平台安全和完整性
- 遵守法律法规要求
- 保护我们用户的权利和安全

## 4. 信息共享和披露

### 4.1 第三方服务集成
- 我们仅在提供集成服务所必需的情况下与第三方服务共享数据
- 数据共享仅限于您通过 OAuth 或 API 连接明确授权的内容
- 我们不向第三方出售或出租您的个人信息

### 4.2 服务提供商
我们可能与协助我们的可信服务提供商共享信息：
- 云托管和基础设施服务
- 分析和监控工具
- 客户支持平台
- 安全和欺诈防护服务

### 4.3 法律要求
在法律要求或我们善意认为披露对以下情况必要时，我们可能披露信息：
- 遵守法律程序或政府要求
- 保护我们的权利、财产或安全
- 防止欺诈或非法活动
- 响应涉及人身安全的紧急情况

### 4.4 业务转让
在合并、收购或资产出售的情况下，您的信息可能转移给收购实体，但需遵守相同的隐私保护。

## 5. 数据安全

### 5.1 安全措施
我们实施行业标准的安全措施，包括：
- **加密**：传输和静态数据加密
- **访问控制**：基于角色的访问和身份验证系统
- **监控**：持续安全监控和威胁检测
- **保险库安全**：使用 HashiCorp Vault 进行安全的凭证存储

### 5.2 第三方安全
- OAuth 令牌和 API 密钥安全存储并加密
- 我们遵循所有第三方集成的安全最佳实践
- 定期安全审计和漏洞评估

### 5.3 数据泄露响应
在发生数据泄露时：
- 我们将迅速调查并控制事件
- 将按法律要求通知受影响的用户
- 我们将根据需要与相关当局合作

## 6. 数据保留

### 6.1 保留期限
- **账户数据**：在您的账户活跃期间以及关闭后的合理期间内保留
- **使用数据**：通常为分析目的保留最多2年
- **日志数据**：为安全和故障排除目的保留最多1年
- **集成数据**：根据您的服务配置和用户偏好保留

### 6.2 删除权利
您可以通过以下方式请求删除您的个人数据：
- 使用我们平台中的账户删除功能
- 直接联系我们提出删除请求
- 请注意，某些数据可能因法律要求或合法商业目的而保留

## 7. 您的权利和选择

### 7.1 访问和控制
- **访问**：请求获取您的个人信息副本
- **更正**：更新或更正不准确的信息
- **删除**：请求删除您的个人信息
- **可携带性**：以可携带格式请求您的数据

### 7.2 通讯偏好
- **营销**：随时选择退出营销通讯
- **通知**：通过您的账户设置控制服务相关通知
- **支持**：就任何隐私相关问题或关切联系我们

### 7.3 Cookie 管理
- **浏览器设置**：通过浏览器设置控制 Cookie
- **选择退出**：使用第三方分析服务提供的选择退出机制
- **必要 Cookie**：请注意，禁用必要 Cookie 可能影响平台功能

## 8. 国际数据传输

### 8.1 跨境传输
您的信息可能传输到并在您所在国家以外的国家处理。我们通过以下方式确保国际传输有适当的保障措施：
- **充分性决定**：传输到具有充分数据保护的国家
- **标准合同条款**：使用欧盟批准的合同保护
- **同意**：在需要时获得您的明确同意

### 8.2 区域合规
我们遵守我们运营地区的适用数据保护法律，包括：
- **GDPR**：欧洲通用数据保护条例
- **CCPA**：加利福尼亚消费者隐私法
- **其他区域法律**：适用于您所在司法管辖区的法律

## 9. 儿童隐私

我们的服务不针对13岁以下的儿童。我们不会故意收集13岁以下儿童的个人信息。如果我们发现我们收集了13岁以下儿童的个人信息，我们将立即采取措施删除此类信息。

## 10. 第三方链接和服务

我们的平台可能包含第三方网站的链接或与外部服务集成。本隐私政策不适用于第三方服务。我们鼓励您查看通过我们平台访问的任何第三方服务的隐私政策。

## 11. 隐私政策变更

### 11.1 政策更新
我们可能不时更新本隐私政策，以反映我们做法或法律要求的变化。我们将通过以下方式通知您重大变更：
- 在我们的网站上发布更新的政策
- 向注册用户发送电子邮件通知
- 提供平台内通知

### 11.2 继续使用
您在政策变更后继续使用我们的服务即构成对更新的隐私政策的接受。

## 12. 特定地区权利

### 12.1 欧盟用户（GDPR）
如果您位于欧盟，您享有额外权利，包括：
- **反对权**：反对基于合法利益的处理
- **限制权**：在某些情况下限制处理
- **投诉权**：向您当地的数据保护机构提出投诉

### 12.2 加利福尼亚用户（CCPA）
如果您是加利福尼亚居民，您享有的权利包括：
- **知情权**：关于收集和共享的个人信息的信息
- **删除权**：请求删除个人信息
- **非歧视权**：因行使您的权利而不受歧视
