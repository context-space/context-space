# 隱私政策

> **重要提示**：本繁體中文版本僅供參考，如有任何衝突或歧義，請以 [英文版本](https://context.space/privacy) 為準。
>
> **Important Notice**: This Traditional Chinese version is for reference only. In case of any conflicts or ambiguities, the English version shall prevail.

**最後更新時間：2025年7月1日**

## 1. 引言

Context Space 致力於保護您的隱私並確保您個人資訊的安全。本隱私政策說明了當您使用我們的上下文工程基礎設施平台時，我們如何收集、使用、共享和保護您的資訊。

透過使用我們的服務，您同意按照本隱私政策所述收集和使用您的資訊。如果您不同意我們的隱私做法，請不要使用我們的服務。

## 2. 我們收集的資訊

### 2.1 您直接提供的資訊
- **帳戶資訊**：姓名、電子郵件地址、使用者名稱和密碼
- **個人資料資訊**：您選擇提供的其他個人資料詳情
- **設定資料**：整合設定、偏好設定和自訂設定
- **支援通訊**：您在聯絡客戶支援時提供的資訊

### 2.2 自動收集的資訊
- **使用資料**：您如何與我們的平台互動、使用的功能和使用頻率
- **技術資料**：IP地址、瀏覽器類型、裝置資訊、作業系統
- **日誌資料**：伺服器日誌、錯誤報告和系統效能指標
- **分析資料**：使用者行為模式和平台效能指標

### 2.3 來自第三方整合的資訊
當您將第三方服務連接到 Context Space 時，我們可能收集：
- **身份驗證權杖**：OAuth 權杖和 API 金鑰用於服務存取
- **服務資料**：您授權我們從連接的服務中存取的資料（GitHub 儲存庫、Slack 訊息、Airtable 記錄等）
- **整合中繼資料**：有關您連接的服務及其設定的資訊
- **同步內容**：透過我們的平台在您連接的服務之間流動的資料

### 2.4 Cookie 和追蹤技術
- **必要 Cookie**：平台功能和安全所需
- **分析 Cookie**：幫助我們了解平台的使用情況
- **偏好 Cookie**：記住您的設定和偏好
- **第三方 Cookie**：來自整合服務和分析提供商

## 3. 我們如何使用您的資訊

### 3.1 服務提供
- 提供和維護我們的上下文工程平台
- 處理和促進第三方服務之間的整合
- 驗證使用者身份並管理帳戶存取
- 啟用資料同步和工作流程自動化

### 3.2 平台改進
- 分析使用模式以改進我們的服務
- 開發新功能和能力
- 監控平台效能和可靠性
- 進行研究和開發活動

### 3.3 通訊
- 發送重要的服務更新和安全通知
- 回應客戶支援詢問
- 提供技術協助和故障排除
- 分享相關的產品更新和改進（經您同意）

### 3.4 安全和合規
- 檢測和防止詐騙活動
- 維護平台安全和完整性
- 遵守法律法規要求
- 保護我們使用者的權利和安全

## 4. 資訊共享和披露

### 4.1 第三方服務整合
- 我們僅在提供整合服務所必需的情況下與第三方服務共享資料
- 資料共享僅限於您透過 OAuth 或 API 連接明確授權的內容
- 我們不向第三方出售或出租您的個人資訊

### 4.2 服務提供商
我們可能與協助我們的可信服務提供商共享資訊：
- 雲端託管和基礎設施服務
- 分析和監控工具
- 客戶支援平台
- 安全和詐騙防護服務

### 4.3 法律要求
在法律要求或我們善意認為披露對以下情況必要時，我們可能披露資訊：
- 遵守法律程序或政府要求
- 保護我們的權利、財產或安全
- 防止詐騙或非法活動
- 回應涉及人身安全的緊急情況

### 4.4 業務轉讓
在合併、收購或資產出售的情況下，您的資訊可能轉移給收購實體，但需遵守相同的隱私保護。

## 5. 資料安全

### 5.1 安全措施
我們實施業界標準的安全措施，包括：
- **加密**：傳輸和靜態資料加密
- **存取控制**：基於角色的存取和身份驗證系統
- **監控**：持續安全監控和威脅檢測
- **保險庫安全**：使用 HashiCorp Vault 進行安全的憑證儲存

### 5.2 第三方安全
- OAuth 權杖和 API 金鑰安全儲存並加密
- 我們遵循所有第三方整合的安全最佳實務
- 定期安全稽核和漏洞評估

### 5.3 資料外洩回應
在發生資料外洩時：
- 我們將迅速調查並控制事件
- 將按法律要求通知受影響的使用者
- 我們將根據需要與相關當局合作

## 6. 資料保留

### 6.1 保留期限
- **帳戶資料**：在您的帳戶活躍期間以及關閉後的合理期間內保留
- **使用資料**：通常為分析目的保留最多2年
- **日誌資料**：為安全和故障排除目的保留最多1年
- **整合資料**：根據您的服務設定和使用者偏好保留

### 6.2 刪除權利
您可以透過以下方式請求刪除您的個人資料：
- 使用我們平台中的帳戶刪除功能
- 直接聯絡我們提出刪除請求
- 請注意，某些資料可能因法律要求或合法商業目的而保留

## 7. 您的權利和選擇

### 7.1 存取和控制
- **存取**：請求獲取您的個人資訊副本
- **更正**：更新或更正不準確的資訊
- **刪除**：請求刪除您的個人資訊
- **可攜帶性**：以可攜帶格式請求您的資料

### 7.2 通訊偏好
- **行銷**：隨時選擇退出行銷通訊
- **通知**：透過您的帳戶設定控制服務相關通知
- **支援**：就任何隱私相關問題或關切聯絡我們

### 7.3 Cookie 管理
- **瀏覽器設定**：透過瀏覽器設定控制 Cookie
- **選擇退出**：使用第三方分析服務提供的選擇退出機制
- **必要 Cookie**：請注意，停用必要 Cookie 可能影響平台功能

## 8. 國際資料傳輸

### 8.1 跨境傳輸
您的資訊可能傳輸到並在您所在國家以外的國家處理。我們透過以下方式確保國際傳輸有適當的保障措施：
- **充分性決定**：傳輸到具有充分資料保護的國家
- **標準合約條款**：使用歐盟核准的合約保護
- **同意**：在需要時獲得您的明確同意

### 8.2 區域合規
我們遵守我們營運地區的適用資料保護法律，包括：
- **GDPR**：歐洲一般資料保護規則
- **CCPA**：加利福尼亞消費者隱私法
- **其他區域法律**：適用於您所在司法管轄區的法律

## 9. 兒童隱私

我們的服務不針對13歲以下的兒童。我們不會故意收集13歲以下兒童的個人資訊。如果我們發現我們收集了13歲以下兒童的個人資訊，我們將立即採取措施刪除此類資訊。

## 10. 第三方連結和服務

我們的平台可能包含第三方網站的連結或與外部服務整合。本隱私政策不適用於第三方服務。我們鼓勵您查看透過我們平台存取的任何第三方服務的隱私政策。

## 11. 隱私政策變更

### 11.1 政策更新
我們可能不時更新本隱私政策，以反映我們做法或法律要求的變化。我們將透過以下方式通知您重大變更：
- 在我們的網站上發布更新的政策
- 向註冊使用者發送電子郵件通知
- 提供平台內通知

### 11.2 繼續使用
您在政策變更後繼續使用我們的服務即構成對更新的隱私政策的接受。

## 12. 特定地區權利

### 12.1 歐盟使用者（GDPR）
如果您位於歐盟，您享有額外權利，包括：
- **反對權**：反對基於合法利益的處理
- **限制權**：在某些情況下限制處理
- **投訴權**：向您當地的資料保護機構提出投訴

### 12.2 加利福尼亞使用者（CCPA）
如果您是加利福尼亞居民，您享有的權利包括：
- **知情權**：關於收集和共享的個人資訊的資訊
- **刪除權**：請求刪除個人資訊
- **非歧視權**：因行使您的權利而不受歧視
